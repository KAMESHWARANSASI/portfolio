About Me

I'm a Java FullStack Developer

I'm Kameshwaran, a passionate and goal-driven Full Stack Developer based in Velachery, Chennai. I hold a Bachelor of Engineering in Electrical and Electronics (EEE) from Mahendra Institute of Technology, Namakkal. Though my academic background is in core engineering, my interest in software development inspired me to become a full-stack developer. I’ve gained hands-on experience in building web applications using Java, Spring Boot, and MySQL for the backend, and HTML, CSS, JavaScript, React, and Bootstrap for the frontend. I also work with Git and GitHub for version control and collaboration. I enjoy creating clean, efficient, and responsive applications that solve real-world problems. Whether it's designing interactive frontends or developing powerful backend systems, I strive to deliver high-quality solutions. I’m currently open to full-time developer roles and available for freelance projects. I'm excited to collaborate with teams or clients who value performance, creativity, and clean code. When I’m not coding, I explore new technologies, contribute to personal projects, and stay active in developer communities.
